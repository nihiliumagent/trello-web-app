# Simple Trello-style Web App

A lightweight, static Kanban board application that works like Trello.

## Features

- **Columns**: Create and manage multiple columns for your workflow
- **Cards**: Add cards to columns with editable text
- **Drag & Drop**: Move cards between columns using drag and drop
- **Persistence**: Your board is automatically saved in localStorage
- **Responsive Design**: Works on desktop and mobile devices
- **No Backend**: Pure static HTML, CSS, and JavaScript

## How to Use

1. Open `index.html` in a web browser
2. Click "+ Add Column" to create new columns
3. Click "+ Add a card" in any column to add cards
4. Drag and drop cards to move them between columns
5. Edit text by clicking on titles or card content
6. Delete columns or cards using the × button

## GitHub Pages Deployment

To deploy this app on GitHub Pages:

1. Create a new repository on GitHub
2. Upload these files to the repository
3. Go to Settings → Pages in your repository
4. Under "Source", select "main" branch and click Save
5. Your site will be live at `https://your-username.github.io/repo-name/`

## Browser Support

- Chrome/Edge (recommended)
- Firefox
- Safari
- Opera

## Notes

- Data is stored in browser localStorage, so it persists between page reloads
- Data is specific to the browser and device where it was created
- No account or login required
